<imported-template>

<imported-template> is a custom element that lets you load HTML partials into your Web Components. See README.md for more information

Examples

<imported-template> features

  • <imported-template> with simple partial
  • <imported-template> from well behaved partial
  • <imported-template> from ill-behaved partial (<iframe>)

<x-html> features

  • <imported-template> in a <template> with HTML partial from a string
  • <imported-template> in a <template> with HTML partial loaded from a file Misbehaved partial - does not contain <template>, but does contain <script>
  • <imported-template> in a <template> with HTML partial loaded directly to DOM tree (not wrapped by <imported-template> node) Misbehaved partial - does not contain <template>, but does contain <script>
  • <imported-template> in a Polymer element

Well behaved partial rules

Composite well-behaved partial

  • Contains <template> to be replicated
  • May contain <script>, <style>, <link> element that are going to be loaded and executed once, on first import

Simplified partial

  • Does not contain any <tempalte>
  • Does not contain any other <script>, <style>, <link>
In this case everything will be moved to a new <template>.

Misbehaved partials

In this case use iframe (<template is="imported-tempalte" iframe>)

Otherwise it may blow up, as entire content will be loaded once as a HTML Import, and then replicated (re-executed) to original document every time.
